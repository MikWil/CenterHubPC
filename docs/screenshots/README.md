# Screenshots

This folder holds screenshots referenced from the main `README.md`.

Drop PNG files here using these exact filenames so the existing README
image refs resolve:

| Filename | What |
|---|---|
| `window-layouts.png` | Window Layouts page with at least one saved layout card visible (Apply / 🔄 / ✏ / 🗑 actions) |
| `update-banner.png` | The sky-blue update banner under the title bar — "Update available — vN.N.N is ready to install" |

Recommended capture:

- Window cropped to the CenterHub frame (drop the OS chrome / taskbar)
- 1360–1500 px wide
- PNG, RGB, no transparency

The README image tags use plain `<img src="docs/screenshots/foo.png">`
relative paths so they render correctly on the GitHub repo view, on the
Releases page, and in cloned local copies.
